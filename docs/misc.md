Miscellaneous registers
=======================

| Register   | Bits          | Description              |
| ---------- | ------------- | ------------------------ |
| `a07001b0` | `00000001` RW | Rumble motor enable      |
| `a07001b0` | `0000007e` RW | Rumble motor speed       |
| `a0730048` | `00000001` R  | Power key (0 if pressed) |
